# JPA 소개

ORM과 JPA

- ORM -> 객체와 RDB 중간에서 매핑역할
- JPA -> 자바 진영의 ORM 기술 표준

JPA를 사용해야 하는 이유

- 객체 중심 개발
- 패러다임 불일치 문제 해결
- 생산성
- 유지보수성
- 성능 최적화
- 데이터 접근 추상화와 벤터 독립성
- 표준

# SQL 중심적인 개발의 문제점

SQL 중심적인 개발의 문제점

- SQL 문의 무한 반복, 지루한 코드
- 엔티티 신뢰성 문제
- 관계형 데이터베이스와 자바 객체의 패러다임 불일치

관계형 데이터베이스와 자바 객체의 차이

- 상속
- 연관관계
- 데이터 타입
- 데이터 식별 방법

결국 다음과 같은 이유로 자바 객체에서 SQL 문으로, SQL 문에서 자바 객체로 변환하는데 개발자가 힘이 든다. 그래서 관계형 데이터베이스에서도 자바 컬렉션처럼 처리할 수 있는 JPA가 등장한다.

# 기타 그래프 이론

서로소 집합 자료구조

- 서로소 부분 집합들로 나누어진 원소들의 데이터를 처리하기 위한 자료구조
- 합집합, 찾기 연산을 지원
- 서로소 집합 자료구조로 사이클 판별도 가능

크루스칼 알고리즘

- 최소 신장 트리를 찾기 위한 대표적인 알고리즘
- 시간 복잡도는 O(NlogN)

위상 정렬 알고리즘

- 사이클이 없는 방향 그래프의 모든 노드를 방향성에 거스르지 않도록 순서대로 나열하는 것
- 시간 복잡도는 O(N+M)

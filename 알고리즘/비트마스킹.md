# 비트마스킹

비트마스킹

- 컴퓨터에서 다루는 최소 단위인 비트를 활용하여 특정 정보를 표현하고 조작하는 기법

비트마스킹 장점

- 메모리 절약
- 빠른 연산
- 간결한 코드
- 집합 표현과 조작

비트마스킹을 이용한 집합 표현

- 원소 추가 -> A |= (1 << K)
- 원소 삭제 -> A &= ~(1 << K)
- 원소 토글 -> A ^= (1 << K)
- 원소의 포함 여부 확인 -> if A & (1 << K)
- 집합 연산

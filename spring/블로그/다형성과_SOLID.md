## 다형성과 SOLID 간단하게 정리

다형성

- 부모 클래스 타입 참조 변수로 자식 클래스 타입의 인스턴스를 참조하는 것, 인터페이스 타입 참조 변수로 구현 객체를 참조하는 것
- 부모 클래스, 인터페이스 타입으로 여러 자식 클래스 타입 인스턴스, 여러 구현 객체를 참조하면서 다양한 결과를 얻을 수 있다.

SOLID

- SRP : 단일 책임 원칙
- OCP : 개방-폐쇄 원칙
- LSP : 라스코프 치환 법칙
- ISP : 인터페이스 분리 법칙
- DIP : 의존관계 역전 법칙

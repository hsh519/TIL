# Spring Data JPA

Spring Data JPA

- 데이터베이스와의 상호작용을 더 쉽고 편리하게 만들어주는 기술
- 직접 구현 클래스를 생성해 프록시 객체를 주입

쿼리 메서드 기능

- 메서드 이름으로 쿼리 생성
- NamedQuery
- @Query
- 반환 타입
- 페이징과 정렬
- 벌크성 수정 쿼리
- @EntityGraph

# 데이터베이스 기본 개념 간단하게 정리

데이터

- 현실 세계에서 측정하거나 관찰해 수집한 값

정보

- 데이터를 유용하게 활용할 수 있도록 처리하여 체계적으로 조직한 결과물

데이터베이스

- 수집한 데이터를 저장하고 필요할 때 제공하는 역할

데이터베이스 정의

- 특정 조직의 여러 사용자가 공유할 수 있도록 통합하고 저장한 운영 데이터

데이터베이스 특징

- 실시간 접근성, 계속 변화, 동시 공유, 내용 기반 참조

데이터 종류

- 정형 데이터, 반정형 데이터, 비정형 데이터

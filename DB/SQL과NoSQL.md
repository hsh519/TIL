## SQL 과 NoSQL 간단하게 정리

SQL

- 행과 열로 구성된 관계형 데이터베이스
- 테이블 구조와 데이터 타입을 사전에 정의
- 스키마에 맞는 데이터만 삽입 가능
- 수직적 확장만 가능
- 데이터 검색, 수정, 삭제, 삽입 가능

NoSQL

- 관계도 없고 정해진 구조도 없는 비관계형 데이터베이스
- 관계형 데이터베이스를 제외한 모든 유형의 데이터베이스
- 대표적인 유형으로는 키-값, 문서, 그래프
- 수직적, 수평적 확장 가능
- 데이터를 읽어올 때 해당 유형에 맞춰서 데이터를 읽어옴
